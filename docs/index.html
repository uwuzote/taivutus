    Indikatiivi &

    Indikatiivin preesens &

    Myönteinen

    Aktiivimuodot
    1. -VA → -V-
    2. -dA → -∅-
    3. -(ll/nn/rr/st)A → -(l/n/r/s)e-
    4. -(A/O/U)ta → -(A/O/U)A-
    5. -itA → -itse-
    6. -etA → -ene-
    Passiivi
    1. 1. tyyppi: -(e/i/O/U)A → -(e/i/O/U)tAAn
      -AA → -etAAn
    2. 2-6. tyypit: -A → -AAn

    Kielteinen

    Aktiivimuodot

    → ei-verbi + kieltomuoto (sama kuin vartalo + KPT-vaihtelu)

    Passiivi

    → ei + myönteinen passiivi ilman viimeistä An:ää

    Esimerkit

    Aktiivimuodot:

    1. oppia → minä opin, hän oppii, hän ei opi
    2. voida → … voin, … voi, … voi
    3. jutella → … juttelen, … juttelee, … juttele
    4. pelätä → … pelkään, … pelkää, … pelkää
    5. tarvita → … tarvitsen, … tarvitsee, … tarvitse
    6. pidetä → … pitenen, … pitenee, … pitene

    Passiivit:

    1. oppia → opitaan, ei opita
      alkaa → aletaan, ei aleta
    2. voida → voidaan, ei voida
    3. jutella → jutellaan, ei jutella
    4. pelätä → pelätään, ei pelätä
    5. tarvita → tarvitaan, ei tarvita
    6. pidetä → pidetään, ei pidetä

    Indikatiivin imperfekti &

    Myönteinen

    Aktiivimuodot
    1. a…-aa (2 tavua) → a…-oi-
      -(l/n/r)tAA* → -(l/n/r)si-
      -(A/e/i)A → -i-
      -(U/O)A → -(U/O)i-
    2. -idA → -i-
      -V₁V₂dA → -V₂i-
    3. -(ll/nn/rr/st)A → -(l/n/r/s)i-
    4. -(A/O/U)ta → -(A/O/U)si-
    5. -itA → -itsi-
    6. -etA → -eni-

    *ja verbit huutaa, löytää, pyytää, tietää, tuntea, taitaa

    Passiivi &
    1. -(e/i/O/U)A → -(e/i/O/U)ttiin
      -AA → -ettiin
    2. -dA → -tiin
    3. -(l/n/r/t)A → -tiin
    4. Verbityypit 4-6.: -tA → -ttiin

    Kielteinen

    Aktiivimuodot

    → ei-verbi + nUt-partisiippi (yksikkö ja kohtelias Te -nUt tai T-monikko -neet)

    Passiivi

    → ei + tU-partisiippi

    Esimerkit

    Aktiivimuodot:

    1. alkaa → minä aloin, hän alkoi
      lentää → … lensin, … lensi
      tuntea → … tunsin, … tunsi
      käyttää → … käytin, … käytti
      uskoa → … uskoin, … uskoi
    2. voida → … voin, … voi
      juoda → … join, … joi
    3. jutella → … juttelin, … jutteli
    4. pelätä → … pelkäsin, … pelkäsi
    5. tarvita → … tarvitsin, … tarvitsi
    6. pidetä → … pitenin, … piteni

    Passiivit:

    1. oppia → opittiin
      alkaa → alettiin
    2. voida → voitiin
    3. jutella → juteltiin
    4. pelätä → pelättiin
    5. tarvita → tarvittiin
    6. pidetä → pidettiin

    Indikatiivin perfekti &

    Aktiivimuodot

    → olla-verbi + nUt-partisiippi (yksikkö ja kohtelias Te -nUt tai monikko -neet)

    Passiivi

    → on / ei ole + tU-partisiippi

    Indikatiivin pluskvamperfekti &

    Aktiivimuodot

    → olla-verbi imperfektimuodossa + nUt-partisiippi (yksikkö ja kohtelias Te -nUt tai monikko -neet)

    Passiivi

    → oli / ei ollut + tU-partisiippi

    Partisiipit &

    Aktiivin partisiipin preesens -vA &

    Akitiivin partiisipin preesens on sama kuin verbin he-muoto indikatiivin preesensissä ilman viimeistä t:tä

    Esimerkit
    1. oppia → oppiva
    2. juoda → juova
    3. purra → pureva
    4. kadota → katoava
    5. tarvita → tarvitseva
    6. vanheta → vanheneva

    Aktiivin partisiipin perfekti -nUt &

    1. -A → -nUt
    2. -dA → -nUt
    3. -(ll/nn/rr/st)A → -(ll/nn/rr/ss)Ut
    4. 4-6. tyypit: -tA → -nnUt
    Esimerkit
    1. oppia → oppinut
    2. juoda → juonut
    3. purra → purrut
    4. kadota → kadonnut
    5. tarvita → tarvinnut
    6. vanheta → vanhennut

    nUt-partisiipin taivutus &

    Ei ole KPT-vaihtelua

    Yksikkö:

    • Partitiivi: -Ut → -UttA
    • Illatiivi: -Ut → -eeseen
    • Muut sijamuodot: -Ut → -ee- + sijapääte

    Monikko:

    • Partitiivi: -Ut → -eitA
    • Genetiivi: -Ut → -ei(d/tt)en
    • Illatiivi: -Ut → -ei(siin/hin)
    • Muut sijamuodot: -Ut → -ei- + sijapääte

    Passiivin partisiipin preesens -tAvA &

    Indikatiivin imperfektin passiivista:

    • -(t/tt)iin → -(t/tt)AvA
    Esimerkit
    1. oppia → opittava
      alkaa → alettava
    2. voida → voitava
    3. jutella → juteltava
    4. pelätä → pelättävä
    5. tarvita → tarvittava
    6. pidetä → pidettävä

    Passiivin partisiipin perfekti -tU &

    Indikatiivin imperfektin passiivista:

    • -(t/tt)iin → -(t/tt)U
    Esimerkit
    1. oppia → opittu
      alkaa → alettu
    2. voida → voitu
    3. jutella → juteltu
    4. pelätä → pelätty
    5. tarvita → tarvittu
    6. pidetä → pidetty

    Agenttipartisiippi -mA &

    Verbin he-muodosta indikatiivin preesensissä:

    • -vAt → -mA
    Esimerkit
    1. oppia → oppima
    2. juoda → juoma
    3. purra → purema
    4. kadota → katoama
    5. tarvita → tarvitsema
    6. vanheta → vanhenema

    Kielteinen agenttipartisiippi -mAtOn &

    Angenttipartisiipista -mA + -tOn

    Esimerkit
    1. oppia → oppimaton
    2. juoda → juomaton
    3. purra → purematon
    4. kadota → katoamaton
    5. tarvita → tarvitsematon
    6. vanheta → vanhenematon

    mAtOn-partisiipin taivutus &

    KPT-vaihtelu on aina tt ← t

    Yksikkö:

    • Partitiivi: -tOn → -tOntA
    • Illatiivi: -tOn → -ttOmAAn
    • Muut sijamuodot: -tOn → -ttOmA- + sijapääte

    Monikko:

    • Partitiivi: -tOn → -ttOmiA
    • Genetiivi: -tOn → -ttOmien (harvoin -tOnten)
    • Illatiivi: -tOn → -ttOmiin
    • Muut sijamuodot: -tOn → -ttOmi- + sijapääte
