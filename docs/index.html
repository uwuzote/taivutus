    Indikatiivin preesens ↑

    Myönteinen

    Persoonamuodot:

    1. -VA → -V-
    2. -dA → -∅-
    3. -(ll/nn/rr/st)A → -(l/n/r/s)e-
    4. -(A/O/U)ta → -(A/O/U)A-
    5. -itA → -itse-
    6. -etA → -ene-

    Passiivi:

    1. -(e/i/O/U)A → -(e/i/O/U)tAAn
      -AA → -etAAn
    2. -A → -AAn

    Kielteinen

    Persoonamuodot: ei-verbi + kieltomuoto (vartalo)

    Passiivi: ei + myönteinen passiivi ilman viimeistä An:ää

    Indikatiivin imperfekti ↑

    Myönteinen

    Persoonamuodot:

    1. a…-aa (2 tavua) → a…-oi-
      -(l/n/r)tAA* → -(l/n/r)si-
      -(A/e/i)A → -i-
      -(U/O)A → -(U/O)i-
    2. -idA → -i-
      -V₁V₂dA → -V₂i-
    3. -(ll/nn/rr/st)A → -(l/n/r/s)i-
    4. -(A/O/U)ta → -(A/O/U)si-
    5. -itA → -itsi-
    6. -etA → -eni-

    *ja verbit huutaa, löytää, pyytää, tietää, tuntea, taitaa
