Verbityyppien preesensin ja imperfektin vartalot

1. verbityyppi -VA

Preseens

  • -VA → -V-

Imperfekti

  • a…-aa → a…-oi-
  • -(l/n/r)tAA* → -(l/n/r)si-
  • -(A/e/i)A → -i-
  • -(U/O)A → -(U/O)i-

*ja verbit huutaa, löytää, pyytää, tietää, tuntea

2. verbityyppi -dA

Yleiset poikkeusverbit: käydä (preesensin var. käy-, imperfektin var. kävi-), nähdä (preesensin var. näke-, imperfektin var. näki-), tehdä (preesensin var. teke-, imperfektin teki-)

Preesens

  • -dA → -∅-

Imperfekti

  • -idA → -i-
  • -V₁V₂dA → -V₂i-

3. verbityyppi -(ll/nn/rr/st)A

Yleinen poikkeusverbi: juosta

Preseens

  • -(ll/nn/rr/st)A → -(l/n/r/s)e-

Imperfekti

  • -(ll/nn/rr/st)A → -(l/n/r/s)i-

4. verbityyppi -(A/O/U)tA

Preseens

  • -(A/O/U)ta → -(A/O/U)A-

Imperfekti

  • -(A/O/U)ta → -(A/O/U)si-

5. verbityyppi -itA

Preesens

  • -itA → -itse-

Imperfekti

  • -itA → -itsi-

6. verbityyppi -etA

Preesens

  • -etA → -ene-

Imperfekti

  • -etA → -eni-
