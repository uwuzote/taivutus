Verbityyppien preesensin ja imperfektin vartalot

1. verbityyppi -VA

Preseens

  • -VA → -V-

Esim. (minä-persoona): puhua → puhun, oppia → opin

Imperfekti

  • a…-aa → a…-oi-
  • -(l/n/r)tAA* → -(l/n/r)si-
  • -(A/e/i)A → -i-
  • -(U/O)A → -(U/O)i-

*ja verbit huutaa, löytää, pyytää, tietää, tuntea, taitaa

Esim. (minä-persoona): maksaa → maksoin, suomentaa → suomensin, tuntea → tunsin, käyttää → käytin, puhua → puhuin

2. verbityyppi -dA

Yleiset poikkeusverbit: käydä (preesensin var. käy-, imperfektin var. kävi-), nähdä (preesensin var. näke-, imperfektin var. näki-), tehdä (preesensin var. teke-, imperfektin teki-)

Preesens

  • -dA → -∅-

Esim. (minä-persoona): juoda → juon

Imperfekti

  • -idA → -i-
  • -V₁V₂dA → -V₂i-

Esim. (minä-persoona): imuroida → imuroin, juoda → join

3. verbityyppi -(ll/nn/rr/st)A

Yleinen poikkeusverbi: juosta (preesensin var. juokse-, imperfektin var. juoksi-)

Preseens

  • -(ll/nn/rr/st)A → -(l/n/r/s)e-

Esim. (minä-persoona): mennä → menen, tulla → tulen, nousta → nousen, purra → puren, jutella → juttelen

Imperfekti

  • -(ll/nn/rr/st)A → -(l/n/r/s)i-

Esim. (minä-persoona): mennä → menin, tulla → tulin, nousta → nousin, purra → purin, jutella → juttelin

4. verbityyppi -(A/O/U)tA

Preseens

  • -(A/O/U)ta → -(A/O/U)A-

Esim (minä-persoona): vastata → vastaan, tykätä → tykkään

Imperfekti

  • -(A/O/U)ta → -(A/O/U)si-

Esim (minä-persoona): vastata → vastasin, tykätä → tykkäsin

5. verbityyppi -itA

Preesens

  • -itA → -itse-

Esim. (minä-persoona): häiritä → häiritsen, valita → valitsen

Imperfekti

  • -itA → -itsi-

Esim. (minä-persoona): häiritä → häiritsin, valita → valitsin

6. verbityyppi -etA

Preesens

  • -etA → -ene-

Esim. (minä-persoona): vanheta → vanhenen, lämmetä → lämpenen

Imperfekti

  • -etA → -eni-

Esim. (minä-persoona): vanheta → vanhenin, lämmetä → lämpenin

Preesensin passiivi

Positiivisessa virkkeessä

Preesensin passiivi on aina heikossa asteessa

1. verbityyppi -VA:

  • -(e/i/O/U)A → -(e/i/O/U)tAAn
  • -AA → -etAAn
2-6. verbityypit:
  • -A → -AAn
Esim.: oppia → opitaan, maksaa → maksetaan, voida → voidaan

Negatiivisessa virkkeessä

1. verbityyppi: 'ei' + positiivinen passiivi ilman viimeistä An:ää
2-6. verbityypit: 'ei' + infinitiivi

Esim.: oppia → opita, maksaa → makseta, voida → voida
