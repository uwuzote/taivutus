Verbityyppien preesensin ja imperfektin vartalot

1. verbityyppi -VA

Preseens

  • -VA → -V-

Imperfekti

  • a…-aa → a…-oi-
  • -(l/n/r)tAA* → -(l/n/r)si-
  • -(A/e/i)A → -i-
  • -(U/O)A → -(U/O)i-

*ja verbit huutaa, löytää, pyytää, tietää, tuntea

2. verbityyppi -dA

Yleiset poikkeusverbit: käydä (preesensin var. käy-, imperfektin var. kävi-), nähdä (preesensin var. näke-, imperfektin var. näki-), tehdä (preesensin var. teke-, imperfektin teki-)

Preesens

  • -dA → -∅-

Imperfekti

  • -idA → -i-
  • -V₁V₂dA → -V₂i-

3. verbityyppi -(ll/nn/rr/st)A

Yleinen poikkeusverbi: juosta

Preseens

  • -(ll/nn/rr/st)A → -(l/n/r/s)e-

Imperfekti

  • -(ll/nn/rr/st)A → -(l/n/r/s)i-

4. verbityyppi -(A/O/U)tA

Preseens

  • -(A/O/U)ta → -(A/O/U)A-

Imperfekti

  • -(A/O/U)ta → -(A/O/U)si-

5. verbityyppi -itA

Preesens

  • -itA → -itse-

Imperfekti

  • -itA → -itsi-

6. verbityyppi -etA

Preesens

  • -etA → -ene-

Imperfekti

  • -etA → -eni-
Preesensin passiivi

Positiivisessa virkkeessä

Preesensin passiivi on aina heikossa asteessa

1. verbityyppi -VA:

  • -(e/i/O/U)A → -(e/i/O/U)tAAn
  • -AA → -etAAn
2-6. verbityypit:
  • -A → -AAn
Esimerkit: oppia → opitaan, maksaa → maksetaan, voida → voidaan

Negatiivisessa virkkeessä

1. verbityyppi: 'ei' + positiivinen passiivi ilman viimeistä An:ää
2-6. verbityypit: 'ei' + infinitiivi

Esimerkit: oppia → opita, maksaa → makseta, voida → voida
